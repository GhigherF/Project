         Для использования программы пользователю требуется вополнить следующие шаги:

1.Скачать архив по ссылке: https://github.com/GhigherF/Project
2.Извлечь архив в рабочую папку
3.Запустить приложение в компиляторе

    После этих действий можно корректно использовать модуль "Калькулятор".
    Для дальнейшего использования модуля "Конвертер валют" пользователю требуется:

1.Скомпилировать проект
2.Найти в папке с проектом файл "1.txt" и скопировать его путь
3.Перевести путь в кодировку Unicode
4.Заменить путь в строке с обьявлением файла
5.Перевести в кодировку Unicode следующее: "curl https://select.by/kurs/ > "PATH" ",где PATH -- путь пользователя к файлу 1.txt
6.Заменить присвоение строки полученными символами

После этого пользователь сможет получить полный функционал проекта

Возможные ошибки:

1.Компилятор может не получить доступ к файлу или командной строке,решается перезапуском приложения с правами администратора
2. В процессе выявления
